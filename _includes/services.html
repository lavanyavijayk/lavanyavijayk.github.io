ATC-Vision – Autonomous Air Traffic Control Agent

Built an autonomous Air Traffic Control agent using LangGraph and Gemini 2.5 Flash, integrating a physics-based simulation engine to model real-time aircraft spawning, movement, and altitude dynamics. Implemented proactive collision prediction and safety checks to detect and resolve flight path conflicts across concurrent aircraft. Validated system robustness by simulating diverse crash scenarios and edge cases to stress-test agent reliability under failure conditions.

MindBridge – Fine-Tuned LLM Evaluation for Mental Health Support

Fine-tuned Llama-3-8B-Instruct using LoRA and QLoRA adapters on real-world therapy conversation datasets to optimize empathetic response generation. Designed an LLM-as-Judge evaluation framework to benchmark fine-tuned model outputs against prompt-engineered large LLMs across clinical quality metrics. Quantified a 43% performance gap between small fine-tuned models and large LLMs, surfacing a critical privacy-performance tradeoff relevant to on-device mental health applications.
